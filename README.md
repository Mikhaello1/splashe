# splashe

Сканер ценовых сплешей на спотовых рынках **MEXC, KuCoin, Gate и Bitget** с
оповещениями в Telegram.

Подключается к публичным WebSocket-потокам сделок всех USDT/USDC-пар на четырёх
биржах, за заданный интервал считает движение цены и объём и, при превышении
порогов, шлёт алерт в Telegram-канал.

## Возможности

- Мониторинг 4 бирж одновременно (можно включать/выключать в `config.json`).
- Алерты в форум-супергруппу Telegram: **отдельный топик под каждую биржу**.
- Кнопка **«Открыть график»** в каждом алерте — ведёт на страницу пары.
- Управление фильтрами прямо из бота (интервал, % движения, объём) — **только для
  админов** из `.env`.
- Настройки хранятся в `config.json` и переживают перезапуск.

## Настройка

1. Создайте бота у [@BotFather](https://t.me/BotFather), получите токен.
2. Создайте супергруппу, включите в ней **Темы (Topics)** и заведите 4 топика —
   под MEXC, KuCoin, Gate и Bitget. Добавьте бота в группу администратором.
3. Скопируйте шаблон и заполните значения:
   ```
   cp .env.example .env
   ```
   - `TELEGRAM_BOT_TOKEN` — токен бота.
   - `ADMIN_IDS` — ваш Telegram ID (узнать у [@userinfobot](https://t.me/userinfobot)),
     несколько — через запятую.
   - `ALERT_CHAT_ID` — ID группы (обычно начинается с `-100`).
   - `TOPIC_MEXC` / `TOPIC_KUCOIN` / `TOPIC_GATE` / `TOPIC_BITGET` — `message_thread_id`
     соответствующих топиков (число в конце ссылки на топик в веб-Telegram).

## Запуск

```
go build -o splash .
./splash
```

При первом запуске рядом появится `config.json` с настройками по умолчанию.

## Фильтры (`config.json`)

| Поле                 | Значение по умолчанию | Описание                                        |
|----------------------|-----------------------|-------------------------------------------------|
| `interval_seconds`   | 1                     | окно анализа в секундах                         |
| `threshold_percent`  | 0.1                   | минимальное движение цены за окно, %            |
| `min_volume_usd`     | 200                   | минимальный объём за окно, USDT                 |
| `alert_cooldown_sec` | 30                    | не слать повторный алерт по паре чаще, чем раз в |
| `quotes`             | `["USDT","USDC"]`     | какие котировки считать долларом                |
| `exchanges`          | все `true`            | какие биржи мониторить                          |

`interval_seconds`, `threshold_percent` и `min_volume_usd` можно менять на лету —
напишите боту `/settings` и жмите кнопки. Изменения сразу применяются и
сохраняются в `config.json`.
